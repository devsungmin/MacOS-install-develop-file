# Install
1. Clone github file
    ``` shell
    git clone https://github.com/devsungmin/MacOS-install-develop-file.git
    ```
2. Add authorization
    ```shell
    chmod +x ./install.sh
    ```
3. Run **install.sh** file
    ```shell
    ./install.sh
    ```
